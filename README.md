# minecraft_rcon

A simple client for sending RCON commands to a Minecraft server.

- [RCON documentation](https://wiki.vg/RCON)

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  minecraft_rcon:
    github: z64/minecraft_rcon
```

## Usage

```crystal
require "minecraft_rcon"
```
